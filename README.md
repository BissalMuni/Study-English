# Study-English — 100일 회화 (하루 1장)

이미지 회화 자료를 **하루에 한 장씩, 100일 동안** 공부하기 위한 저장소.

## 폴더 구조

```
Study-English/
├── images/Day100/
│   ├── Day100_001.jpg ~ Day100_100.jpg   # 메인 회화 이미지 100장 (하루 1장)
│   └── pic/                              # 추가 그림 (100 1 ~ 100 5, 각 20장)
├── notes/                               # 하루치 학습 노트 (day-001.md ...)
└── README.md                            # 이 파일 — 100일 진도 체크리스트
```

## 사용법

1. 하루에 `Day100_NNN.jpg` 한 장씩 공부한다.
2. 끝내면 아래 체크리스트에 `[x]` 표시한다.
3. 필요하면 `notes/day-001.md` 처럼 그날 배운 표현을 정리한다.
