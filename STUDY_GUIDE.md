# AI로 100일 회화 노트 200% 활용하기

`notes/DayNNN.md` 한 장을 AI(Claude·ChatGPT 등)에게 붙여넣고, 아래 프롬프트를 복사해 쓰면 됩니다.
핵심 원칙은 단 하나 — **읽지 말고 "출력"하라.** AI를 _대화 상대·시험관·코치_ 로 세워서 말하기/쓰기를 강제하는 게 능률의 핵심입니다.

---

## 0. 하루 학습 루틴 (15~20분)

| 단계 | 시간 | 무엇을 | AI 활용 |
|---|---|---|---|
| ① 입력 | 3분 | 오늘 노트(`DayNNN.md`) 표현·대화 읽기 | — |
| ② 말하기 | 5분 | 대화를 소리 내어 따라 하기(쉐도잉) | **프롬프트 5 (발음 코치)** |
| ③ 출력 | 5분 | 한국어만 보고 영어로 말/쓰기 | **프롬프트 3 (한→영 드릴)** |
| ④ 응용 | 5분 | 표현을 내 상황에 바꿔 쓰기 | **프롬프트 2 (상황 변형)** |
| ⑤ 복습 | 2분 | 지난 며칠치 섞어 퀴즈 | **프롬프트 4 (간격복습 퀴즈)** |

> 매일 ①②만 해도 됩니다. 시간 날 때 ③④⑤로 깊이를 더하세요.

## 1. 롤플레이 — AI를 대화 상대로
```
아래는 내가 공부하는 영어 회화 노트야. 너는 B 역할을 맡아 나(A)와 자연스럽게
영어로 대화해 줘. 내가 어색하게 말하면 한 번 더 자연스러운 표현을 알려주고 이어가 줘.
대화가 끝나면 내가 쓴 표현 중 고칠 점 3가지만 짚어 줘.

[여기에 DayNNN.md 내용 붙여넣기]
```

## 2. 상황 변형 — 같은 표현, 새 상황
```
이 노트의 핵심 표현을 그대로 써서, 노트와 "다른 상황" 3개의 짧은 대화를 만들어 줘.
난이도는 (초급 / 중급 / 실전) 으로 하나씩. 영어 + 한국어 해석 같이.

[DayNNN.md 붙여넣기]
```

## 3. 한→영 작문/스피킹 드릴 (가장 효과 큼)
```
이 노트의 대화에서 "한국어 문장만" 하나씩 보여 줘. 나는 영어로 답할게.
내 답을 노트의 정답과 비교해서 ① 맞았는지 ② 더 자연스러운 대안 ③ 자주 하는 실수
순으로 피드백해 줘. 한 문장씩 진행하자.

[DayNNN.md 붙여넣기]
```

## 4. 간격 복습(spaced repetition) 퀴즈
```
Day{N-6} ~ Day{N} 노트 7개의 핵심 표현을 섞어서, 빈칸 채우기 5문제 +
"이 상황에 뭐라고 말할까?" 4지선다 5문제로 퀴즈를 내 줘.
내가 다 풀면 채점하고, 틀린 표현만 다시 3문제 더 내 줘.

[해당 Day 노트들 붙여넣기]
```

## 5. 발음·쉐도잉 코치
```
이 대화의 각 영어 문장에 대해 ① 강세 위치 ② 연음(linking)·축약되는 부분
③ 한국인이 자주 틀리는 발음 포인트를 알려 줘. 끝나면 쉐도잉용으로
천천히/원어민 속도 두 버전의 "끊어 읽기" 표시를 해 줘.

[DayNNN.md 붙여넣기]
```

## 6. 표현 확장 — 격식/비격식·유사·반대
```
이 노트의 핵심 표현에 대해 ① 더 격식 있는 버전 ② 더 캐주얼한 버전
③ 비슷한 뜻 표현 3개 ④ 반대 상황에서 쓰는 표현을, 각각 짧은 예문과 함께 정리해 줘.

[DayNNN.md 붙여넣기]
```

## 7. 나만의 예문 만들기 (개인화 = 기억에 오래 남음)
```
나는 [직업/관심사: 예) 회사원, 등산 좋아함] 야. 이 노트의 핵심 표현을 써서
"내 실제 일상"에 쓸 법한 예문 5개를 만들어 줘. 영어 + 한국어.
그중 가장 자연스러운 1개를 골라 이유도 알려 줘.

[DayNNN.md 붙여넣기]
```

## 8. 약점 추적 — 틀린 것만 모아 재출제
```
지금까지 내가 자주 틀린 표현들이야: [표현 나열].
이걸로만 미니 테스트(한→영 5문제)를 내 줘. 다 맞히면 통과, 틀리면
약점 노트에 추가할 수 있게 한 줄 요약을 만들어 줘.
```

- 어려웠던 표현은 `notes/_review.md`(직접 생성) 같은 파일에 모아 두고 **프롬프트 8**로 주기적으로 복습하세요.
- 일주일에 한 번은 **프롬프트 4**로 7일치를 통째로 섞어 테스트 → 진짜 기억에 남았는지 확인.

## 핵심 요약
1. **출력 강제** — 읽기 < 말하기·쓰기. AI를 시험관으로.
2. **개인화** — 내 일상 예문으로 바꿔야 안 잊어버린다.
3. **간격 복습** — 오늘 것 + 며칠 전 것을 섞어라.
4. **즉시 교정** — 틀리면 그 자리에서 자연스러운 표현으로 받기.
